Plugin dùng để tạo nút chat Zalo và một số tính năng đi kèm

== Cài đặt ==
1. Upload thư mục "linhna_chat_zalo" vào thư mục "/wp-content/plugins/".
2. Kích hoạt plugin thông qua trang 'Plugins' trong WordPress.

== Hướng dẫn sử dụng ==
Dùng shortcode sau đặt vào bất cứ trang nào muốn hiển thị nút chat Zalo:
[linhna_zalo id="snh_zalo_chat"]
